# **App Name**: AquaPoints

## Core Features:

- Student Profile: Students create profiles, view Hydro-Point balance, and track hydration goals.
- Hydro-Station Locator: Display Hydro-Stations via a map interface with walking directions.
- Hydration Challenges: AI engine generates gamified challenges to earn Hydro-Points and create habits.
- Swipe Card Integration: Integrate the physical swipe card system to register hydration activities.
- Leaderboard & Rewards: Show a leaderboard ranking based on points accumulated, tied to in-app rewards.
- Admin Dashboard: Admins track hydration trends, manage Hydro-Stations, and distribute Hydro-Points, including tracking usage and errors.
- Personalized Recommendations: Generative AI creates and offers tailored recommendations and tips based on daily progress; Gemini acts as a tool providing personalized content that is displayed within the app

## Style Guidelines:

- Primary color: Light blue (#ADD8E6), reflecting a water theme for a refreshing feel.
- Background color: Very light blue (#F0F8FF), almost white, creating a clean and airy aesthetic.
- Accent color: Deep blue (#00008B), used sparingly for key interactive elements to draw focus.
- Clean, card-based layout for easy navigation and data presentation.
- Body and headline font: 'PT Sans', a humanist sans-serif font that mixes a modern look with a little warmth.
- Use water droplet icons with increasing fills to represent hydration levels.
- Subtle animations and transitions when updating Hydro-Point balance and displaying encouragement messages.
